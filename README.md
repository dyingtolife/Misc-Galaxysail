## 概览

Misc-Galaxysail 是一款面向 CTF 竞赛 MISC 方向的集成解题工具平台，专为隐写分析、编码解码、压缩包加密分析三大核心领域设计。工具以单文件 exe 形式分发，无需安装 Python 或任何运行环境依赖，解压即可使用，开箱即用。

> 本工具处于持续开发阶段，将根据 CTF 赛事趋势和用户反馈不断新增功能模块与分析能力，敬请关注后续更新。

### 设计定位

- **目标用户**：CTF 参赛者、安全研究人员、隐写分析爱好者
- **运行方式**：双击 exe 启动本地 Web 服务（默认 `http://127.0.0.1:7890`），通过浏览器操作所有功能
- **支持格式**：PNG / JPG / BMP / GIF / ICO / WebP / TIFF 图片 + ZIP / RAR / 7Z / TAR 压缩包 + TXT / LOG / DATA / MD 文本文件
- **文件限制**：单张图片最大 100MB，文本文件无大小限制，字典文件无大小限制

### 三大核心模块

| 模块 | 功能 |
|------|------|
| **图片隐写分析** | 6 个独立分析模块：元数据与文本块提取、文件拼接检测与分离、LSB 位平面分析（54 种组合）、GIF 逐帧差异分析、PNG/JPEG 宽高 CRC 爆破修复、FFT 盲水印与双图像素运算 |
| **字母/文本分析** | 20+ 种编码解码（Base64/32/58、Hex、二进制、八进制、URL、HTML 实体、Unicode 转义）+ 7 种古典密码 + Magic 自动识别 + 递归解码 + Base64 隐写提取 + 十六进制编辑器 |
| **ZIP 加密分析** | ZIP/RAR/7Z/TAR 格式检测与结构解析、伪加密一键修复、CRC32 碰撞破解释文、字典攻击、掩码暴力破解、文件头/中央目录/EOCD 结构修复、批量处理 |

### 技术特色

- **完全离线运行**：所有分析在本地完成，无需联网，不上传任何文件到外部服务器
- **单文件分发**：打包为独立 exe，无 Python 运行时依赖
- **授权保护**：设备绑定授权机制，一机一码
- **Web UI 操作**：现代化响应式界面，操作直观
- **结果持久化**：所有分析结果自动保存为 JSON 格式，支持搜索、导出、统计看板
- **批量处理**：支持 ZIP 批量字典/掩码破解，流式实时反馈破解进度
- **容错性强**：支持损坏图片、不完整压缩包的分析与修复

## 授权激活

工具首次运行需要激活，激活后永久有效，重启自动完成激活。

激活信息保存在 exe 所在目录的 `instance/license.key` 文件中。只要该文件存在，启动工具时就会自动检测并完成激活，无需重复输入授权码。

### 激活流程

**1. 获取机器码**

打开激活页面，页面会显示当前机器的机器码（MAC 地址），例如：

```
机器码: 985F413CB617
```

**2. 发送机器码获取授权码**

将上述机器码发送给工具作者，作者会根据机器码生成对应的授权码并返回给你。

**3. 输入授权码**

将生成的授权码复制到激活页面输入框，点击「激活」按钮。激活成功后页面会显示授权文件路径。

**4. 验证激活**

导航栏显示已激活状态，可正常使用所有功能。

## 功能详述

### 1. 图片隐写分析

独立页面：`http://127.0.0.1:7890/image-analysis`

支持格式：PNG、JPG/JPEG、BMP、GIF、ICO、WebP、TIFF  
单文件上限：100MB

上传图片后，可手动选择 6 个分析模块逐一执行分析，每个模块独立查看结果。

#### 元数据与文本块分析

读取图片元数据与文件格式信息。

- **基本信息**：尺寸、文件大小、色彩模式
- **EXIF 信息**：拍摄设备、GPS 定位、时间戳、软件信息、缩略图
- **PNG 文本块**：tEXt、zTXt、iTXt 等文本块中的隐藏注释
- **文件尾部数据**：检测 PNG 的 IEND 之后或 JPEG 的 FFD9 之后是否有追加数据
- **二维码检测**：自动扫描图片中的二维码并解码内容
- **关键字搜索**：在文件所有可读字符串中搜索 `flag{}`、`CTF{}`、MD5/SHA1/SHA256 哈希值

#### 文件合并/分离检测

识别图片尾部拼接的其他文件。

- **Magic 签名检测**：通过文件魔数（Magic Number）自动识别拼接的 ZIP、RAR、7z、GZ、PDF 等格式，执行文件雕刻
- **文件提取**：从图片中分离出拼接的压缩包和其他文件，保存到 `results/` 目录
- **偏移量定位**：精确显示每个拼接文件的起始偏移位置和大小

#### LSB 位平面分析

执行 LSB（最低有效位）隐写检测。

- **54 种组合扫描**：遍历 RGB 三通道、不同位平面的所有组合
- **三种视图模式**：
  - ASCII 模式：以文本形式显示提取内容
  - HEX 模式：以十六进制字节显示
  - SMART 模式：智能过滤不可读字符，仅展示有意义的内容
- **正则搜索**：在 LSB 提取内容中即时搜索 `flag{...}`、`ctf{...}` 等关键字
- **匹配导航**：逐条定位匹配项，自动滚动到对应 LSB 组合
- **多种位深度**：支持 1-bit、2-bit、4-bit、8-bit 提取

#### GIF 帧分析

针对 GIF 动图进行逐帧分析。

- **逐帧拆解**：将 GIF 动图拆分为单帧图片并逐帧预览
- **帧差异比较**：计算相邻帧之间的像素差异，高亮显示变化区域
- **增强帧提取**：对存在差异的帧进行增强处理，便于查看隐藏内容
- **帧导出**：支持导出每一帧为独立图片文件

#### 宽高与 CRC 修复

检测 PNG/JPEG 图片的宽高异常，自动修复 CRC 并爆破正确宽高。

- **CRC 校验**：验证 PNG 的 IHDR 块 CRC 校验值是否与当前宽高匹配
- **CRC 自动修复**：当 CRC 与当前宽高不匹配时，自动写入正确的 CRC 值
- **宽高爆破**：遍历可能的宽高组合，找到使 CRC 匹配的正确尺寸
- **异常检测**：标记宽高值异常或被篡改的图片
- **修复输出**：生成修复后的图片并提供下载链接

#### 盲水印/双图叠加分析

支持单图和双图两种工作模式。

**单图模式：**
- 显示图片基本信息（尺寸、色彩模式等）
- 提供盲水印检测所需的前提信息

**双图模式（上传两张图片后）：**
- **FFT 盲水印解码**：从两张图片中比对提取水印
- **像素运算**：支持 XOR（异或）、ADD（叠加）、SUB（相减）等图片运算
- **差异区域检测**：自动检测并高亮两张图片之间的差异区域
- **多策略提取**：同时尝试多种盲水印提取策略并展示所有结果
- **结果预览与下载**：支持在线预览提取结果，提供 PNG 格式下载

### 2. 字母/文本分析

独立页面：`http://127.0.0.1:7890/text-decoder`

支持上传 TXT/LOG/DATA/MD 文件，也支持直接粘贴文本内容进行分析。提供左右对比模式：原文在左，解码结果在右。

除此之外，还提供单独的十六进制编辑器页面，支持对文件进行十六进制级别的查看和修改。

#### 编码解码

| 类型 | 说明 |
|------|------|
| **Base64** | 编码与解码，自动检测 Base64 字符串 |
| **Base32** | 编码与解码 |
| **Base58** | 编码与解码 |
| **十六进制 (Hex)** | 编码与解码 |
| **二进制** | 二进制转文本解码 |
| **八进制** | 八进制转文本解码 |
| **URL 编码** | URL 百分比编码与解码 |
| **HTML 实体** | HTML 实体编码与解码（`&amp;`、`&#xx;` 等） |
| **Unicode 转义** | `\uXXXX` 格式的 Unicode 转义序列解码 |

#### 古典密码

| 类型 | 说明 |
|------|------|
| **凯撒密码** | 编码解码 + 全部 25 种偏移的暴力破解 |
| **ROT13** | ROT13 编码与解码 |
| **ROT47** | ROT47 编码与解码 |
| **Atbash** | Atbash 密码编码与解码 |
| **栅栏密码** | 编码解码 + 全部栏数的暴力破解 |
| **摩斯密码** | 摩斯电码解码 |
| **培根密码** | 培根密码解码 |

#### 智能工具

**Magic 自动解码**  
输入一段未知编码的文本，系统自动尝试所有编码方式（Base64、Hex、URL、Base32、二进制等），智能识别可读结果，返回所有有意义的解码结果。

**递归解码**  
逐层解码直到无法继续，完整展示解码链路。例如 `Base64 → Hex → ROT13 → 明文`，每一层的中间结果和使用的解码方式都清晰可见。

**Flag 提取**  
自动扫描文本中的 `flag{...}`、`ctf{...}`、MD5(32位十六进制)、SHA1(40位十六进制)、SHA256(64位十六进制) 等格式，一键定位关键信息。

**Base64 隐写提取**  
从 Base64 编码的 `=` 填充位中提取隐藏数据，适用于 Base64 隐写术（Base64 Steganography）题型。

#### 文本工具

- **字符串反转**：将文本内容反转
- **大小写转换**：全大写 / 全小写转换
- **去除空白字符**：移除文本中的所有空白字符
- **正则提取**：使用自定义正则表达式从文本中提取匹配内容

#### 二进制数据处理

当解码结果为二进制数据（如 ZIP 压缩包、图片等非纯文本数据）时，系统自动：

1. 将二进制数据保存到 `results/decoded_bin/` 目录
2. 显示文件的保存路径
3. 生成十六进制转储预览表（包含偏移量、十六进制、ASCII 三列）
4. 提供下载链接

#### 十六进制编辑器

提供完整的十六进制编辑功能，支持对上传的文件进行字节级别的查看和修改。

- **文件加载**：以十六进制方式加载任意文件，显示偏移量、十六进制值、ASCII 对照三列视图
- **字节修改**：支持在指定偏移位置修改字节值
- **全量写入**：支持插入或追加字节后全量写入文件（适用于文件大小发生变化的场景）
- **另存文件**：将修改后的文件保存到本地目录，不覆盖原文件

### 3. ZIP 加密分析

独立页面：`http://127.0.0.1:7890/zip-analysis`

支持格式：ZIP、RAR、7Z、TAR 以及从其他文件（如图片）尾部提取的压缩包数据块

#### 格式检测

- **自动检测**：通过文件魔数自动识别 ZIP / RAR / 7Z / TAR 格式
- **容错识别**：即使文件损坏或不完整，也能通过特征签名识别
- **数据块提取**：从其他文件中提取嵌入的 ZIP 数据块（指定偏移量）

#### 结构分析

**ZIP 分析：**
- 解析 ZIP 文件头、中央目录、EOCD
- 列出压缩包内所有文件及其状态（加密 / 伪加密 / 普通）
- 统计文件总数、加密文件数、压缩比
- 检测加密算法类型（ZipCrypto / AES）

**RAR 分析：**
- RAR 格式检测与版本识别（RAR4 / RAR5）
- 文件列表与属性查看
- 伪加密标志位检测

**7Z 分析：**
- 7Z 格式检测
- 内容列表查看
- 伪加密检测

**TAR 分析：**
- TAR 格式检测
- 文件列表查看

#### 伪加密检测与修复

- **ZIP 伪加密**：检测文件头的 bit-6 加密标志位是否与全局标志位不一致，一键修复为正常压缩包
- **RAR 伪加密**：检测并修复 RAR 文件头中的加密标志位
- **7Z 伪加密**：检测并修复 7Z 头部中的加密标志位

#### CRC32 碰撞破解

通过 CRC32 校验值反推明文密码。适用于：
- 已知 CRC32 值的加密小文件
- 已知部分明文内容的加密文件
- 自动遍历常见字符组合进行碰撞

#### 字典攻击

- 使用字典文件逐条尝试密码
- 支持上传自定义字典文件
- 支持 ZIP 标准加密和 AES 加密
- 支持 RAR 字典攻击
- 实时显示破解进度和当前尝试的密码

#### 掩码攻击（暴力破解）

- 按指定模式暴力破解密码
- 可选字符集：数字、小写字母、大写字母、全部字母数字、自定义字符集
- 可自定义密码最小长度和最大长度
- 实时显示破解进度

#### 批量处理

- **批量扫描**：一次性扫描压缩包内所有文件的状态（加密/伪加密/普通）
- **批量字典破解**：同时对多个加密文件进行字典攻击，自动处理每个文件
- **批量掩码破解**：同时对多个加密文件进行掩码攻击
- **流式反馈**：实时推送每个文件的破解进度、状态更新和结果，支持逐条展示
- **单文件重试**：针对破解失败的文件独立调整参数重新尝试

#### 结构修复

- **ZIP 文件头修复**：修复损坏或偏移的文件头签名
- **中央目录重建**：当中央目录损坏时，从本地文件头重建
- **EOCD 修复**：修复损坏的中央目录结尾记录
- **数据描述符修复**：处理缺失或损坏的数据描述符

#### 破解历史

系统自动记录已破解的密码（Potfile），避免重复破解。可通过页面查看所有历史破解记录。

## 结果管理

工具自动保存所有分析结果，支持完整的增删查改操作。

### 结果页面

访问 `http://127.0.0.1:7890/results` 查看所有历史分析结果。

### 结果详情

每条结果包含：
- 任务 ID（唯一标识）
- 使用的工具名称
- 分析的文件名和 MD5 哈希
- 所有发现项列表（含置信度标记：高 / 中 / 低）
- 发现数量统计
- 创建时间

### 操作功能

- **搜索**：按文件名、工具名、任务 ID 搜索历史结果
- **分页浏览**：支持分页参数控制每页显示数量
- **导出 JSON**：将所有发现项导出为结构化 JSON 文件
- **导出 CSV**：将结果摘要导出为 CSV 表格（含任务 ID、工具、文件名、发现数、置信度等）
- **删除单条**：删除不需要的历史结果
- **清空全部**：一键清空所有结果

### 统计看板

结果页面顶部显示统计看板：
- 总结果数
- 总发现项数
- 高置信度发现数
- 近 7 天新增结果数
- 按工具分类统计
- 按状态分类统计

### 存储路径

所有结果以 JSON 格式存储在以下位置：

```
results/
├── results_index.json      # 结果索引（元数据 + 统计）
├── files/
│   └── result_*.json       # 每条结果的详细数据
├── decoded_bin/            # 文本解码产生的二进制文件
└── <analysis_id>/          # 图片分析步骤产物
```

## 常见问题

**Q: 如何获取授权码？**

A: 将激活页面显示的机器码发送给工具作者，作者会根据机器码生成授权码并返回给你。

**Q: 图片分析支持哪些格式？**

A: 支持 PNG、JPG/JPEG、BMP、GIF（含动图帧分析）、ICO、WebP、TIFF，单张最大 100MB。双图模式下最多同时上传 2 张图片进行盲水印分析。

**Q: 分析结果保存在哪里？**

A: 所有分析结果保存在 `results/` 目录下，以 JSON 格式存储。图片分析产生的提取文件按分析 ID 组织子目录。结果页面会显示每个文件的保存路径。

**Q: 如何清理临时文件？**

A: 调用 `POST /cleanup` 接口可清理 `uploads/` 和 `results/temp/` 目录的临时文件。

**Q: 如何重新激活？**

A: 删除 exe 所在目录下的 `instance/license.key` 文件，重启 exe 即可进入激活页面重新激活。

**Q: ZIP 破解需要多长时间？**

A: 取决于密码复杂度和破解方式。伪加密可在几秒内修复；CRC32 碰撞适用于短密码（4-6 位）；字典攻击取决于字典大小；掩码攻击取决于密码长度和字符集大小。系统会实时显示破解进度。
